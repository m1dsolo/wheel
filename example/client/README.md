## Introduction

A simple tui client written in python.

See [chat_server](../chat_server) for more details.

## Usage

Connect to the server(127.0.0.1:12345) by running the following command:
```bash
python example/client/client.py 127.0.0.1 12345
```

Key bindings:
- `tab`: switch between input and output
- `q`: quit when you are in the output window
- `d`: toggle dark mode when you are in the output window
